# NorthAssist AI

NorthAssist AI is an AI-powered community support navigator designed to help Northern Ontario residents discover government benefits, understand official documents, and find relevant community resources.

## Features

### Benefit Finder
Identify government programs and assistance based on user profile.

### Document Explainer
Upload government letters and receive plain-language explanations.

### Community Resource Search
Search local support programs and services.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI
- OpenAI API

## Project Goal

Build an accessible AI assistant that improves access to benefits, services, and information for Northern Ontario communities.
